# Firefighting Robot
**Bachelor of Science in Mechanical Engineering - Senior Design Project (2019)**

## 🤖 Project Overview
This was a collaborative senior design project focused on developing a firefighting robot capable of climbing stairs and reaching higher locations of fire. It uses water as the fire extinguishing agent.

## 🛠️ Technical Specifications
* **Locomotion:** High-traction rubber tracks driven by brushed DC Motors and 10-teeth metal drive sprockets.
* **Actuation Systems:** A custom Lifting System powered by Worm Gear Motors and supported by linear rods, tension springs, and mini V-wheels.
* **Sensing & Control:**
    * Positioning control using limit switch sensors specifically for the lifting mechanism.
    * Remote control and motor controllers for robot movement.
* **Fluid Delivery:** Integrated pump system for extinguishing fire.
* **Cooling System** Fans for extended operation of microcontrollers and actuators.
